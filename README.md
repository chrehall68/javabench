# JavaBench

Benchmarking classic Data Structures and Algorithms implemented in Java

## Features

- Sorting algorithms
    - MergeSort
    - QuickSort
    - Radix
    - ShellSort
- Data Structures:
    - Heaps
    - AVL Trees
    - Linked Lists
    - Queues
    - Hash Maps
- Unit Tests - implemented for all the sorting algorithms
- Benchmarks - coming soon, currently just in Main.java
